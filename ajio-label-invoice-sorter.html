Marketplace Tool · AJIO Packing Workflow

AJIO Label Invoice Sorter

Arrange AJIO labels and invoices by forward bag barcode, stamp Seller SKU ID and Forward Consignment Bar Code on each label, then download one packing-ready PDF.

How it works

  • Upload AJIO bulk accepted Excel, Label PDF and Invoice PDF.
  • The tool automatically finds the Order Details sheet and reads Customer Order ID, Seller SKU ID and Forward Consignment Bar Code.
  • It matches label pages and invoice pages using AJIO order number / AWB barcode data.
  • Final PDF is generated in selected sequence. Default is bag barcode sorting for faster packing desk arrangement.

Privacy: No uploads. Everything stays on your device.

Built for AJIO sellers who print in batches

This tool is useful when AJIO gives three files during order processing: one bulk accepted Excel file, one label PDF and one invoice PDF. It combines the three into one clean dispatch PDF and keeps the packing sequence aligned with the scanned bag barcode series.

Recommended use

  • Scan or arrange AJIO bags in barcode series.
  • Upload the same accepted Excel, label PDF and invoice PDF.
  • Keep default sorting as Forward Consignment Bag Barcode.
  • Print the final PDF and pack in the same bag barcode sequence.
